# 🚀 LangChain Framework Learning Project

This repository contains practical examples and experiments created while learning the **LangChain framework**. The project demonstrates important LangChain concepts such as **messages, middleware, model integration, structured output, and tools** using Python and Jupyter notebooks.

It serves as a **learning resource for developers** who want to understand how to build applications with LangChain.

# 📘 File Description

### main.py
Entry point of the project with a simple Python script.

### requirements.txt
Contains all required Python dependencies.

### langchainintro.ipynb
Introduction to LangChain and its basic concepts.

### 4-messages.ipynb
Demonstrates how **messages work in LangChain**, including user messages and system messages.

### 6-middleware.ipynb
Explains how **middleware works in LangChain pipelines**.

### ModelIntegration.ipynb
Shows how to integrate **different language models with LangChain**.

### structured_output.ipynb
Demonstrates how to generate **structured outputs using schemas**.

### tools.ipynb
Shows how **tools can be integrated with agents** to extend LLM capabilities.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/langchain-framework.git
cd langchain-framework
```

Create virtual environment

```bash
python -m venv venv
```

Activate virtual environment

### Windows

```bash
venv\Scripts\activate
```

### Mac / Linux

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Running the Project

Run the main script

```bash
python main.py
```

Explore the notebooks for learning LangChain step by step.

Start Jupyter Notebook

```bash
jupyter notebook
```
